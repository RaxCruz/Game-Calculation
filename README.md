# 心算遊戲: 挑戰右腦力 V1.0

## 介紹:
自己有在練習珠心算, 加上網路上又沒有類似概念的心算遊戲, 透過自己的想法打造一個專屬的練習網站.

**遊戲模式**
|-|加法|減法|乘法|除法|
|-|---|---|---|---|
|間隔|1.5s|1.5s|1.5s|1.5s|
|筆數|7筆|7筆|7筆|7筆|
|轉場|無|脈搏模式|無|無|

## 未來版本:
1. 新增排行榜模式 加入**競技場**
1. 新增RWD, 使手機可以享受更好的遊玩品質
